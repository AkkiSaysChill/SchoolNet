# Project for ai vidyaseetu 1.0 National Round

## School Network Application

A Flask-based school network application with user authentication, social features, admin moderation, and AI-powered academic assistance.

### Features

- User registration and authentication
- Profile management with picture uploads
- Social feed with posts and comments
- Admin dashboard for content moderation
- AI chatbot for academic doubt-solving
- Responsive design with modern UI

### Setup

1. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

2. Set up the database:

   ```bash
   python app.py
   ```

   The database will be created automatically.

### Running the Application

```bash
python app.py
```

Visit `http://localhost:5000` in your browser.

### AI Chatbot

- api key maine hata di project uske bina chatbot kaam nahi kerega, mujhse mang lena

The chatbot provides helpful responses for academic questions. If no API key is configured, it uses intelligent fallback responses covering:

- Mathematics
- Science subjects
- Study tips and strategies
- General academic guidance
